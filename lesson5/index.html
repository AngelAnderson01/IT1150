Internet Engineering Task Force

The Internet Enginerring Task Fource (IETF) is a large open international community of network designers, operators, vendors, and researchers concerned with the evolution of the Internet architecture and the smooth operation of the Internet.[1]

The IETF recently improved the TLS 1.3 protocol, also known as the Transport Layer Security protocol. It delivers superior privacy, secuirty, and performance over the previous versions.[2]

Internet Corporation for Assigned Names and Numbers

ICANN is a not-for-profit public-benefit corporation with participants from all over the world dedicated to keeping the Internet secure, stable and interoperable.[3]

Every wonder how you are able to type in a websites name and it goes directly to it? Well you can thank ICANN for being able to type the name instead of the IP address of that website. ICANN helps link the websites name to it's IP address; since that is how computers on the network know it. [4]

ICANN created the Registrant Program which helps raise awareness about the issues and challenges registrants face.[5]

World Wide Web Consortium

The World Wide Web Consortium (W3C) is an international community where member organizations, full-time staff, and the public work together to develop Web standards.
W3C's mission is to lead the Web to its full potential. [6]

The W3C launched the Merchant Business Group. Which is an open forum to address challanges for customers experiences and business needs using Web technologies. [7]
